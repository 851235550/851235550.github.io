无知是这个世界最可怕的事情

发表于 2017-03-25

今天翻看电影《建党伟业》发现了一个问题：电影中叙述1919年第一次世界大战结束，巴黎和会上
由于北洋政府之前与日本秘密签订的二十一条，被日本强占山东半岛，消息传回国内，积压已久的国耻
情绪爆发,随引发了学生等的游行示威。

在顾维钧等代表中国政府参加巴黎和会之前，有一个镜头是顾维钧和梁启超的对话，大意是梁启超觉的
中国作为战胜国一方应该可以得到一部分利益，但是顾维钧表示很难并说了一句话：弱国无外交。

弱国无外交，当时的政府不是不想收回山东半岛，只是由于很多政治原因，没有能力去做这件事。试想
一下，处在当时的中国，南方打完,北方打，北方打完，南北打,覆巢之下，焉有完卵。

爆发游行的原因有很多，我觉的最大的原因并不是学生爱国。而是其中有很多人想借此机会搞点事情。
而学生由于属于最先接触到新思想的一个群体，所以成了最容易发动的群体。

提到上面这些内容，并不是想洗地。而是觉的经过一个世纪的发展，无知的群众并没有因为时代的进步
而减少。反而随着人口基数的扩大，信息流通的方便等原因到处都能见到无知的现象。

什么样的情况下才会进化出有独立思想的群体呢？或者人需要经历什么样的事情才会觉醒？不要人云亦
云，这是我们从小受到的教育。但我们同样在接受要服从组织的教育。我们甚至都不能去直面我们过去
的失败，言论受到极大的束缚。

最近发现了自己的一个很大的问题，总是把自己摆在很高的位置，对别人尤其是觉的那些经验不足的人
总是一副过来人的样子去教育别人。造成这种现象的原因我觉的是因为我觉的自己比别人优秀，自己在
心中觉的高人一等。但是，反过来想我也知道我自己并没有什么东西可以拿来秀优越啊。转念一想就明
白了，和自己觉的经验不足的人比，我的确有很多地方比他们强。但是觉的自己没有东西秀优越，是拿
自己和更优秀的人比，这就解了这个问题。

穷则独善其身，达则兼济天下。我不是这样的人，我可以做到明哲保身，却做不到兼济天下。观念的不
同，后天生活环境的不同才造就了今天的我们。很多时候我们与其抱怨，不如静下心来想一想该如何解
决目前的问题；想一想造成这种问题的原因，或许我们就会有所领悟。

永远不要把自己的想法强加给别人。我们每个人兴趣不同，生活的环境不同，使我们每个人形成了不同
的三观。如果没有长时间的相处，就不要对别人妄加猜测。
